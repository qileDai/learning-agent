# 古诗词鉴赏方法

## 鉴赏步骤（四步法）

1. **读懂字面**：扫清生字、典故障碍
2. **体会意境**：意象组合营造的画面与氛围
3. **把握情感**：诗人写作背景与抒情方式
4. **品味语言**：炼字、修辞、音韵格律

## 常见意象

| 意象 | 常见含义 |
|------|----------|
| 月 | 思乡、离别、团圆 |
| 柳 | 送别、留恋 |
| 雁 | 书信、思乡 |
| 梅 | 高洁、坚贞 |
| 菊 | 隐逸、傲霜 |

## 抒情方式

- **直抒胸臆**：直接表达情感，如「人生自古谁无死」
- **借景抒情**：以景写情，如「枯藤老树昏鸦」
- **托物言志**：借物喻人，如《爱莲说》

## 修辞手法

- **比喻、拟人、夸张** — 形象生动
- **对偶** — 格律诗常见，形式整齐
- **用典** — 增添文化底蕴，需结合注释理解

## 代表篇目速记

### 李白
- 《静夜思》思乡
- 《将进酒》豪放不羁

### 杜甫
- 「诗史」反映安史之乱民生疾苦
- 《春望》国破家亡之痛

### 苏轼
- 《念奴娇·赤壁怀古》咏史抒怀
- 豪放词派代表
